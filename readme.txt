== Description ==
 
This plugin will update the WordPress password hahsing algorithm to implement the new Argon2i algorithm for more security within WordPress.

Existing user accounts will have their password hash updated with Argon2i on the next successful sign in.
 
== Installation ==
 

- Upload the plugin zip to WordPress.

- Activate the plugin.
